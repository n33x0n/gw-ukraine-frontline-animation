## 🇵🇱 Polski

### Opis
Projekt **gw-ukraine-frontline-animation** to interaktywna mapa i animacja wideo przedstawiająca przebieg linii frontu w Ukrainie na przestrzeni czasu. Aplikacja oparta jest na bibliotece Leaflet.js i korzysta z pobieranych danych GeoJSON (m.in. z DeepStateMap).

### Instalacja (z GitHuba)
Aby uruchomić projekt na swoim dysku, sklonuj repozytorium:

```bash
git clone https://github.com/n33x0n/gw-ukraine-frontline-animation.git
cd gw-ukraine-frontline-animation
```

Zaleca się uruchomienie projektu za pomocą lokalnego serwera HTTP (np. wtyczka *Live Server* w VS Code, Node.js `http-server` lub Python `python -m http.server`), aby poprawnie wczytywały się dane i pliki JavaScript.

## 🇬🇧 English

### Description
The **gw-ukraine-frontline-animation** project is an interactive map and video animation showing the changes in the frontline in Ukraine over time. The application is built using the Leaflet.js library and relies on GeoJSON data.

### Installation (from GitHub)
To run the project locally, clone the repository:

```bash
git clone https://github.com/n33x0n/gw-ukraine-frontline-animation.git
cd gw-ukraine-frontline-animation
```

It is recommended to run the project via a local HTTP server (e.g., the *Live Server* extension in VS Code, Node.js `http-server`, or Python `python -m http.server`) to ensure all data and JavaScript files are loaded correctly.
